## 📈 Key Findings (2021 Highway Analysis)
**High-Risk Conditions**  
✅ Clear weather: **24,757** crashes (highest)  
✅ Non-intersection: **26,622** crashes  
✅ Daytime peak 13:00: **2,166** crashes  

**Geographic Hotspots**  
✅ Texas: **3,513** highway crashes (#1)  
✅ California: **3,484** highway crashes (#2)  

**Temporal Patterns**  
✅ Sunday: **6,339** crashes (highest)  
✅ Urban highways: **57.5%** (20,102 crashes)  

**Driver Factors**  
✅ DUI: **25.8%** (9,121 highway crashes)  

**2021 Totals**  
✅ Crashes: **35,413** | Fatalities: **38,449**  
✅ Q4 peak: **9,561** crashes, **10,364** deaths
